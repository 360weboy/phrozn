Phrozn Project: PEAR channel

Using this Channel

This channel is to be used with the PEAR installer.

Registering the channel: 
pear channel-discover pear.phrozn.info
Listing available packages: 
pear remote-list -c phrozn
Installing a package: 
pear install phrozn/package_name
Installing a specific version/stability: 
pear install phrozn/package_name-1.0.0
pear install phrozn/package_name-beta
Receiving updates via a feed: 
http://pear.phrozn.info/feed.xml
